# DemoECommerce.ProductApiSolution

![.NET](https://img.shields.io/badge/-.NET-blue?logo=.net&logoColor=white)

## 📝 Description

DemoECommerce.ProductApiSolution is a robust and scalable backend service built using the .NET framework, specifically designed to handle the core product catalog and inventory management for a modern e-commerce ecosystem. This solution provides high-performance RESTful APIs that enable seamless integration for product searching, categorization, and data management. Engineered with a focus on clean architecture and performance, it serves as a reliable foundation for delivering rich product experiences across web and mobile platforms.

## 🛠️ Development Setup

### .NET Setup
1. Install [.NET SDK](https://dotnet.microsoft.com/)
2. Restore dependencies: `dotnet restore`
3. Build the project: `dotnet build`
4. Run the project: `dotnet run`
